## The Core Idea

This framework reinterprets space as a flowing substance with a fundamental speed \(c\). Matter, gravity, light, and time are all different aspects of this flow:

- **Space** is the only fundamental substance. It flows at speed \(c\).
- **Matter** is a stable local “speed-difference structure” within this flow.
- **Gravity** is the gradient of that speed difference.
- **Light** is a wave in that gradient.
- **Time** is the direction in which speed differences dissolve.

## Why GPS?

GPS satellites orbit at about 20,200 km altitude. Their atomic clocks run faster than on the ground by about 38 µs/day. This correction is already explained by relativity, but it can also be seen as:

- The **speed difference** \(c - v\) between the satellite and space.
- The **gradient** of that speed difference due to Earth’s gravity.

Both effects modify the satellite’s internal clock, because the clock is a speed-difference structure. The fact that we must correct for this daily is operational proof that space is not empty – it is a flowing medium with speed \(c\).
